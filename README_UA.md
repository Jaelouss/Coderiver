# Coderiver

[Read in English](./README.md)

[**Живий перегляд**](https://jaelouss.github.io/Coderiver/) | [**Репозиторій**](https://github.com/Jaelouss/Coderiver)

Сучасний React додаток, створений на Vite та TypeScript, з вражаючими 3D анімаціями частинок та преміум UI дизайном.

![Скріншот головної сторінки](./public/mainPageScreenshot.png)

## Технічний стек

- **Core:** React 19, TypeScript, Vite
- **Візуал:** Three.js, tsparticles, GSAP, Styled Components
- **Ефекти:** use-scramble

## Особливості

- **Хвиля частинок:** Кастомна, продуктивна 3D анімація хвилі на Three.js з динамічною амплітудою та щільністю.
- **Плавна взаємодія:** Ефекти "scramble text" та відшліфовані стани наведення.
- **Сучасний UI:** Кастомні SVG та повна адаптивність.
- **Готовність до продакшену:** Налаштований ESLint, аліаси шляхів та оптимізована збірка.
